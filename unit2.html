Chapter 2: Number System, Conversion, and Boolean Logic

2.1 Number Systems

A number system defines a set of values used to represent quantity. In computer science, several number systems are commonly used:

  • Decimal (Base-10): Uses digits 0-9. It's the standard system for denoting integer and non-integer numbers.
  • Binary (Base-2): Uses digits 0 and 1. It's the fundamental language of computers.
  • Octal (Base-8): Uses digits 0-7. It was used in earlier computing systems.
  • Hexadecimal (Base-16): Uses digits 0-9 and letters A-F. It's often used in programming and computing to simplify binary representation.

2.1.1 Conversion Between Number Systems

Understanding how to convert numbers between different bases is crucial in computer science.

Decimal to Binary Conversion

To convert a decimal number to binary:

  • Divide the decimal number by 2.
  • Record the remainder.
  • Continue dividing the quotient by 2 until you reach 0.
  • The binary representation is the sequence of remainders read from bottom to top.

Example: Convert (13)10 to binary.

Division by 2 Quotient Remainder
13 ÷ 2 6 1
6 ÷ 2 3 0
3 ÷ 2 1 1
1 ÷ 2 0 1

Reading the remainders from bottom to top: (13)10 = (1101)2

Binary to Decimal Conversion

To convert a binary number to decimal, multiply each bit by 2 raised to the power of its position (starting from 0) and sum the results.

Example: Convert (1101)2 to decimal.

(1 × 23) + (1 × 22) + (0 × 21) + (1 × 20) = 8 + 4 + 0 + 1 = 13

So, (1101)2 = (13)10

Hexadecimal to Binary Conversion

Each hexadecimal digit corresponds to a 4-bit binary sequence.

Example: Convert (A9)16 to binary.

  • A = 10 = (1010)2
  • 9 = (1001)2

So, (A9)16 = (10101001)2

Binary to Hexadecimal Conversion

Group the binary digits into sets of four (starting from the right). Convert each group to its hexadecimal equivalent.

Example: Convert (10101001)2 to hexadecimal.

  • (1010) = A
  • (1001) = 9

So, (10101001)2 = (A9)16

2.2 Complement Systems

Complement systems are used in computing to simplify the subtraction operation and to represent negative numbers.

2.2.1 1's Complement

The 1's complement of a binary number is obtained by inverting all bits (changing 0 to 1 and 1 to 0). This is used for representing negative numbers in certain systems.

Example: The 1's complement of (1010)2 is (0101)2.

2.3 Boolean Logic

Boolean logic is a branch of mathematics that deals with binary values, true and false, represented as 1 and 0. It is essential in the design of digital circuits and programming.

2.3.1 Basic Boolean Operators

  • AND: The result is true (1) only if both operands are true. (A AND B = 1 if A = 1 and B = 1).
  • OR: The result is true if at least one operand is true. (A OR B = 1 if A = 1 or B = 1).
  • NOT: Inverts the value of the operand. (NOT A = 1 if A = 0, and NOT A = 0 if A = 1).

2.3.2 Truth Tables

Truth tables show all possible outputs for a given set of inputs based on Boolean operations.

A B A AND B A OR B NOT A
0 0 0 0 1
0 1 0 1 1
1 0 0 1 0
1 1 1 1 0

Boolean logic is fundamental for designing logic gates used in electronic circuits.
