Introduction to Multimedia

Multimedia combines 'Multi' (many) and 'Media' (ways of expressing information). Thus, multimedia is expressing information in different forms such as text, audio, video, graphics, and animation for better understanding and communication.

Integrating multimedia components using computer systems enhances communication effectiveness. Multimedia systems use both hardware (printer, webcam, microphone) and software (MS Word, PowerPoint, Photoshop) to integrate these components, for example, creating videos by combining audio and images.

Advantages/Merits/Pros/Importance of Multimedia

  • Enhances communication effectiveness.
  • Improves topic understanding.
  • Makes games, cartoons, and movies more realistic.
  • Enhances teaching and learning intuitiveness.
  • Increases realism in virtual reality and simulations.

Disadvantages/De-merits/Cons/Limitation of Multimedia

  • Multimedia components can be expensive.
  • Requires technical knowledge and IT literacy.
  • Technology is constantly evolving.
  • Multimedia devices require electricity.

Components/Elements of Multimedia

Text

Collection of alphanumeric characters, forming words, lines, or paragraphs. Customizable in size, color, style using editors like MS-Word, Notepad. Basic form of information expression. Common extensions: .docx, .txt.

Graphics / Images

Pictures or visual representations. A picture conveys more than text. Quality defined by pixels (resolution). Edited with software like Adobe Photoshop, CorelDraw. Common extensions: .jpeg, .png.

Audio

Sound component, providing hearing sensation. Enhances understanding. Includes human voice, music, synthesized sounds. Transferred digitally. Played using software like Windows Media Player. Common extensions: .mp3, .wma.

Video

Moving visuals created from rapid display of static images (frames). Smoothness measured in Frames Per Second (FPS). Standard FPS rates: 30, 60. Edited using software like iMovie, Filmora. Common extensions: .mp4, .mpeg.

Animation

Sequence of vector (computer-generated) images for visual effects. Used for complex ideas, entertainment, movies. Can be 2D or 3D (more realistic). Created with software like Maya, AutoCAD, Flash. Common extensions: .flv, .gif.
